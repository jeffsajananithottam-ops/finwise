💰 FinWise - Personal Finance Dashboard

FinWise is a modern and responsive Personal Finance Management Dashboard built using HTML, CSS, JavaScript, and Chart.js. It helps users track their income, expenses, budgets, savings goals, and transaction history through an intuitive and interactive interface.

Designed with an Indian financial context, all monetary values are displayed in Indian Rupees (₹).


 🚀 Features

### 📊 Dashboard Overview
- View Total Income
- View Total Expenses
- Track Current Balance
- Monitor Savings Progress
- Interactive Spending Analytics Chart

💸 Expense Management
- Add expenses with:
  - Category
  - Amount
  - Date
  - Notes
- Delete expenses
- View recent expense history

 💰 Income Tracking
- Record income sources
- Track earnings over time
- Delete income entries

📈 Budget Planning
- Set monthly budget limits by category
- Compare spending against budgets
- Monitor remaining budget balance

 🎯 Savings Goals
- Create savings goals
- Set target amounts
- Track current savings progress
- Update savings contributions
- Delete completed or unwanted goals

📜 Transaction History
- Unified view of income and expenses
- Search and filter transactions
- Chronological transaction records

🌙 Dark Mode Support
- Toggle between Light and Dark themes
- Improved accessibility and user experience

👤 Multi-User Authentication
- Login/Register functionality
- User-specific financial data storage
- Persistent local storage support

 📱 Responsive Design
- Desktop friendly
- Tablet compatible
- Mobile optimized navigation
